# vr-folded6

folded 6 by Maeda Mameo for XR with three.js

**Live demo**: https://code4fukui.github.io/vr-folded6/

## Features

- 3d visualization
- 3D rendering with Three.js

## Technology Stack

**Browser modules (ES imports):**

- [`three.module.js`](https://code4fukui.github.io/three.js/build/three.module.js)
- [`ARButton.js`](https://code4fukui.github.io/three.js/examples/jsm/webxr/ARButton.js)


## Installation & Setup

No build step required. Clone the repository and open `index.html` in a browser,
or visit the live demo at https://code4fukui.github.io/vr-folded6/
