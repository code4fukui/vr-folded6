# vr-folded6
日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

A WebXR-enabled Three.js project that displays a 3D folded diamond crystal structure.

## Demo
A live demo is available at [mameson.com/experiment/glsl/folded_6/folded_6.html](http://mameson.com/experiment/glsl/folded_6/folded_6.html).

## Features
- Procedurally generated 3D diamond crystal structure
- WebXR support for AR/VR viewing
- Shader-based rendering using Three.js

## Requirements
- A WebXR-enabled browser (e.g. Google Chrome, Mozilla Firefox)
- A VR headset or AR-capable device (optional)

## Usage
1. Open the `index.html` file in a WebXR-enabled browser.
2. If prompted, allow access to the device's AR/VR capabilities.
3. The 3D scene will be displayed, and you can interact with it using your mouse, keyboard, or VR/AR controls.
