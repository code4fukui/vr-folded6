# vr-folded6
日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

WebXR対応のThree.jsプロジェクトで、3Dの折り畳まれたダイヤモンド結晶構造を表示します。

## デモ
ライブデモは[mameson.com/experiment/glsl/folded_6/folded_6.html](http://mameson.com/experiment/glsl/folded_6/folded_6.html)で利用できます。

## 機能
- 3Dダイアモンド結晶構造の手続き的生成
- AR/VRビューイングのためのWebXRサポート
- Three.jsを使用したシェーダーベースのレンダリング

## 要件
- WebXRに対応したブラウザ (例: Google Chrome、Mozilla Firefox)
- VRヘッドセットまたはAR対応デバイス (オプション)

## 使用方法
1. WebXR対応のブラウザで `index.html` ファイルを開きます。
2. 要求された場合は、デバイスのAR/VR機能へのアクセスを許可します。
3. 3Dシーンが表示され、マウス、キーボード、またはVR/AR操作を使ってインタラクションできます。
